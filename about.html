About Eorzea Wardrobe

Eorzean Personal Wardrobe is a static website designed for FF14 players to showcase and manage character glamour outfits.

Browse job-categorized clothing through an intuitive interface, view detailed equipment information, and permanently preserve your fashion creations.

Key features

  • Static, lightweight site — easy to host (GitHub Pages, Netlify, Vercel).
  • Job-based galleries (Offense, Tank, Heal, All) with detail pages for each outfit.
  • Local-data approach — outfit pages are static HTML (editable), no backend required.
  • Responsive layout and optimized images for fast browsing.

How to edit

Edit the static detail pages (for example all-detail1.html) or add new detail files under the project root. Images are stored under images/ and static data are kept in simple HTML/JSON files.
